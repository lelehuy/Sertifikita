# Fonts

Letakkan font TTF/OTF custom di sini, misal:
- `Poppins-Regular.ttf`
- `GreatVibes-Regular.ttf`

Di aplikasi, pada panel kanan bagian **Font TTF** klik **TTF...** dan pilih font
yang kamu taruh di folder ini.

Tips:
- Gunakan font lisensi bebas (Google Fonts) atau pastikan kamu punya izinnya.
